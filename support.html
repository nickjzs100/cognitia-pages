Cognitia Support

Practice smarter by listening, speaking, and reviewing.

Cognitia is a study app for iPhone and iPad that helps you practice history, geography, and science questions with spoken reading, microphone answers, and flexible round controls.

What Cognitia Does

  • Practice history, geography, and science questions
  • Listen to questions read aloud
  • Answer by microphone
  • Reveal or cover clues as you practice
  • Track score during a round
  • Import your own text packets from the Files app

Supported Devices

  • iPhone
  • iPad
  • Recommended: iOS/iPadOS 17.0 or later

Troubleshooting

Microphone does not work

  • Make sure you tapped Answer by Mic.
  • Allow microphone access in Settings > Privacy & Security > Microphone.
  • Allow speech recognition in Settings > Privacy & Security > Speech Recognition.
  • Confirm the device has a working network connection if Apple speech services are needed by the system.

Packet import does not work

  • Make sure the file is plain text.
  • Select the file through the system Files picker.
  • Check that the packet text is in the format expected by Cognitia.

Question reading does not play

  • Turn the device volume up.
  • Make sure the device is not muted.
  • Tap Start Reading.
